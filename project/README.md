# نظام تتبع المصروفات الصوتي (Voice Expense Tracker)

تطبيق ويب ذكي لتسجيل المصروفات باستخدام الصوت، مبني باستخدام Python Flask.

## المميزات
- 🎙️ **تسجيل صوتي**: تحويل الكلام إلى نص واستخراج المبلغ والوصف تلقائيًا.
- 📊 **لوحة تحكم**: رسوم بيانية وإحصائيات للمصروفات.
- 📱 **تصميم متجاوب**: واجهة عصرية تعمل على جميع الأجهزة (Tailwind CSS).
- 💾 **تخزين Excel**: حفظ البيانات في ملف Excel محلي لسهولة الإدارة.
- 🌍 **دعم العربية**: واجهة عربية كاملة (RTL).

## المتطلبات
- Python 3.8+
- FFmpeg (لمعالجة الصوت)

## التثبيت والتشغيل

1. **تثبيت المكتبات المطلوبة**:
   ```bash
   pip install -r requirements.txt
   ```

2. **التأكد من وجود FFmpeg**:
   يجب أن يكون FFmpeg مثبتًا ومضافًا إلى مسار النظام (System PATH).
   للتحقق: `ffmpeg -version`

3. **تشغيل التطبيق**:
   ```bash
   python app.py
   ```

4. **الاستخدام**:
   - افتح المتصفح على: `http://127.0.0.1:5000`
   - اضغط على زر الميكروفون وتحدث (مثال: "صرفت 500 جنيه في سوبر ماركت").
   - راجع النص المستخرج واضغط "حفظ".
   - للدخول للوحة التحكم:
     - الرابط: `/admin/login`
     - المستخدم: `admin`
     - كلمة المرور: `admin`

## هيكلة المشروع
- `app.py`: ملف التطبيق الرئيسي (Flask).
- `services/`: يحتوي على منطق التعامل مع Excel ومعالجة الصوت.
- `templates/`: ملفات HTML.
- `static/`: ملفات CSS و JS.
- `data/`: مجلد حفظ ملف Excel.

## ملاحظات
- يستخدم التطبيق Free Google Speech API. للأداء العالي (Production)، يفضل استخدام Google Cloud API Key أو Whisper المحلي.
- لتغيير كلمة مرور الأدمن، قم بتعديل `app.py`.
